# 🌍 English Excellence — Language School Landing Page

English Excellence is a modern, responsive landing page developed for an interactive English language learning platform. Crafted by **Team Atlas**, this project seamlessly implements Figma UI/UX design into a clean, accessible, and fast-loading web interface using modern web technologies.

---

## ✨ Project Overview

- **Pixel-Perfect Implementation**: Faithful translation of responsive design layouts across mobile, tablet, and desktop viewports.
- **Interactive UI/UX**: Includes an intuitive mobile navigation menu, course pricing cards, application form validation, and student reviews.
- **Modern Build Pipeline**: Powered by Vite for ultra-fast local development and efficient production asset bundling.

---

## 🛠️ Tech Stack

- **Markup & Styling**: HTML5, Modern CSS3, PostCSS (`postcss-sort-media-queries`, `modern-normalize`)
- **Tooling & Plugins**: Vite, `vite-plugin-html-inject`, `vite-plugin-full-reload`
- **Scripting**: Vanilla JavaScript (ES6+)

---

## 🚀 Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/atlas-team-04.git
   cd atlas-team-04
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
